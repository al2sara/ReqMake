# ReqMake
Advanced Request Sending Tool

## Features
+ Full support for GET, POST, PUT, PATCH, DELETE, OPTIONS methods.
+ HTTP/s, Socks4, Sock5 proxy support.
+ Custom request headers - body.
+ Custom content type.
+ Output received headers and body.
